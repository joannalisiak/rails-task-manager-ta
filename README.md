## :hammer_and_wrench: Setup
1. `git clone git@github.com:joannalisiak/rails-task-manager-ta.git`
2. `cd rails-task-manager-ta`
3. `bundle install`
4. `rails s`
5. Go to `localhost:3000/tasks` in your browser
